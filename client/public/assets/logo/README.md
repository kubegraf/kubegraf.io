KubeGraf.io Logo Assets
========================

This directory contains **public logo assets** for the KubeGraf.io marketing site.

Structure
---------

- `kubegraf_full_color.png` – primary full‑color KubeGraf logo used in the main navbar/header.

Usage
-----

In React components, reference the public logo using a root‑relative URL:

```tsx
<img src="/assets/logo/kubegraf_full_color.png" alt="KubeGraf" />
```

This path is served from the Vite `public` directory (`client/public`).
