Features

Everything you need to manage Kubernetes clusters

Terminal UI

Vim-style keyboard navigation. Browse pods, deployments, and services without leaving your terminal.

Web Dashboard

Real-time WebSocket updates, interactive topology graphs, and port-forward management.

Topology Graphs

D3.js force-directed visualization showing relationships between resources.

Live Metrics

Real-time CPU, memory, and pod status with sparkline charts.

Security Analysis

Automated scanning for SecurityContext, RBAC, and network policy best practices.

Multi-Namespace

Switch contexts and namespaces instantly. View across your entire cluster.

Installation

Get up and running in seconds

macOS 

brew install kubegraf/tap/kubegraf

Linux 

curl -sSL kubegraf.io/install.sh | bash

Windows 

scoop install kubegraf
