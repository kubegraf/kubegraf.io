Quick Install (Recommended)

curl -sSL https://kubegraf.io/install.sh | bash

Automatically detects your architecture and installs the latest version. Requires curl and tar.

Manual Download

# Download from GitHub Releases
wget https://github.com/kubegraf/kubegraf/releases/latest/download/kubegraf-linux-amd64.tar.gz
tar -xzf kubegraf-linux-amd64.tar.gz
sudo mv kubegraf /usr/local/bin/

PowerShell (Recommended - One-Liner)

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser -Force; irm https://kubegraf.io/install.ps1 | iex

This command sets execution policy and installs KubeGraf automatically. Requires PowerShell 5.1+.

Alternative: Download Script (If one-liner doesn't work)

Invoke-WebRequest -Uri https://kubegraf.io/install.ps1 -OutFile install.ps1; powershell -ExecutionPolicy Bypass -File install.ps1

Downloads the script and runs it with bypassed execution policy (no system changes required).
