Brain & Performance

🧠

Next-gen intelligence + unified state

Brain Panel V1 brings a local-only SRE brain, while the new global store speeds every resource tab with smart caching.

Brain Panel (V1)

Accessible via the header toggle, the Brain drawer surfaces a 24–72h timeline of incidents, spikes, and scaling events plus an AI-ready summary JSON.

  • Cluster timeline (incidents, warnings, scaling)
  • OOM + reliability insights with metric cards
  • Rule-based recommendations when AI services are absent

Global Store + Cache

A new SolidJS store centralizes cluster selection, multi-namespace state, and theming, persisting choices to localStorage.

  • Resource cache keyed by cluster + namespaces
  • Pods/Deployments/Services use 15s TTL and background refreshes
  • Skeleton-first UX with gentle updates to avoid flicker

How You Work

Three Ways to Interact

Choose your interface—terminal, web, or modern SPA—all synced in real-time.

Terminal UI

Keyboard-first terminal interface with vim-style navigation. Browse pods, deployments, and services without leaving your shell. SSH-friendly and works over low-bandwidth connections.

$ kubegraf ⎈ KubeGraf v1.0.0 Pods · Deployments · Services · Nodes ● nginx-7bf98d · Running · 12% ● api-server-5c9 · Running · 45% ● worker-batch · Pending

Vim keybindings · Zero dependencies · Works over SSH

Web Dashboard

Browser-based dashboard with real-time WebSocket updates. View metrics, logs, and events in unified views. Perfect for team collaboration and presenting cluster health.

Real-time synchronization between all interfaces ensures consistency across your workflow.

Modern SPA

Customizable workspace with saved filters, favorite resources, and shareable views. Built-in runbooks help standardize incident response across teams.

Drag-to-customize layouts and export diagnostics for postmortems.
